# Phase 10: 홈 화면 UX 강화 및 외부 트렌드 콘텐츠 추가 (7단계 중 9단계)

---

Phase 10에서는 홈 화면이 허전하게 느껴지는 문제를 해결하고,  
검색 이전 단계에서도 사용자가 탐색할 수 있는 트렌드 콘텐츠를 제공하여  
앱의 체류 시간과 재방문율을 높입니다.

본 Phase는 **UI 정렬 안정화 → 홈 콘텐츠 확장** 순서로 진행합니다.

---

## 0. 구현 목표 요약

- 홈 상단 UI 정렬 문제 해결 (검색창 / 옵션 / 버튼)
- 실시간 인기 검색어를 “칩 형태 + 2줄 배치”로 개선
- 검색 이전에 볼 수 있는 홈 하단 추천 콘텐츠 추가
  - 유튜브 인기 영상
  - 트렌딩 뉴스 기사
- 모든 콘텐츠는 클릭 시 팝업 형태로 열림

## 1단계: 홈 상단 UI 안정화 (우선 구현)

---

### 1-1) 실시간 인기 검색어 UI 개선

#### 기존 문제
- 인기 검색어가 가로 전체를 차지하여 밀도가 낮음
- 한 줄에 하나씩만 표시되어 공간 활용이 비효율적

#### 개선 방향
- 버튼(칩) 형태로 변경
- 글자 길이에 맞게 자동 크기 조정
- 가로로 여러 개 배치
- 최대 2줄까지 표시

#### UI 규칙
- 한 줄당 최대 6개
- 총 최대 12개 노출
- 클릭 시 해당 키워드로 즉시 검색 실행

#### UI 텍스트
- 섹션 제목: `🔥 실시간 인기 검색어`
- 빈 상태:
  - `아직 검색 기록이 없습니다. 키워드를 입력해 첫 검색을 시작해보세요!`

---

### 1-2) 검색 영역 레이아웃 정렬 개선

#### 기존 문제
- 옵션 버튼이 검색 입력창보다 위로 떠 보임
- 검색 버튼이 지나치게 길어 조작성이 떨어짐
- 시각적으로 버튼 간 균형이 맞지 않음

#### 개선 방향
- 검색 입력창 + 옵션 버튼을 한 줄에 정렬
- 검색 버튼은 폭을 줄여 명확한 클릭 영역 제공
- 옵션 버튼은 검색 입력창의 **오른쪽 끝**에 배치

#### 권장 레이아웃 구조
- 1행:
  - 검색 입력창 (좌)
  - 옵션 버튼 (우)
- 2행:
  - 검색 버튼 (고정 폭, 과도하게 길지 않게)

#### 버튼 텍스트 (한글 고정)
- 검색 버튼: `🔍 뉴스 검색`
- 옵션 버튼: `⚙️ 조건`

## 2단계: 홈 하단 추천 콘텐츠 영역 추가

---

### 2-1) 홈 하단 섹션 구성

검색을 하지 않아도 볼 수 있는 콘텐츠 영역을 홈 하단에 추가합니다.

#### 섹션 제목
- `📌 오늘의 추천 콘텐츠`

#### 구성
- 유튜브 인기 영상 카드
- 트렌딩 뉴스 기사 카드

각 영역은 **4열 그리드**로 구성하며, 한 줄에 4개씩 표시합니다.

---

## 3단계: 유튜브 인기 영상 카드

---

### 3-1) 데이터 소스

- YouTube Data API 사용
- 인기 영상(Trending) 기준
- API Key 필요 (Google Cloud Console 발급)

---

### 3-2) 카드 구성 요소

각 카드에 포함되는 정보:
- 영상 썸네일 이미지
- 영상 제목(2줄 제한)
- 재생 버튼 또는 “보기” 버튼

---

### 3-3) 레이아웃 규칙

- 4열 그리드
- 최대 8개 노출 (2줄)
- 썸네일 비율 유지

---

### 3-4) 클릭 동작 (팝업)

- 카드 클릭 시 팝업(dialog) 열림
- 팝업 내부에 YouTube iframe 임베드
- 닫기 버튼 제공

#### 팝업 제목 예시
- `📺 인기 유튜브 영상`

---

## 4단계: 트렌딩 뉴스 기사 카드

---

### 4-1) 데이터 소스

- 기존 뉴스 검색 인프라 활용
- 트렌딩 기준:
  - 최근 24시간 인기 키워드 기반 검색
  - 또는 주요 뉴스 도메인에서 최신 기사

> 외부 검색어 트렌드 API에 의존하지 않음

---

### 4-2) 카드 구성 요소

- 기사 썸네일(가능한 경우)
- 기사 제목
- 언론사 이름 또는 도메인

---

### 4-3) 레이아웃 규칙

- 유튜브 카드와 동일한 4열 그리드
- 최대 8개 노출

---

### 4-4) 클릭 동작 (팝업)

- 카드 클릭 시 팝업 열림
- 팝업 내부 구성:
  - 기사 제목
  - 간단 요약
  - “🔗 새 탭에서 기사 보기” 링크

⚠️ iframe 웹 임베드는 사이트 정책상 차단될 수 있으므로  
기본 동작은 **새 탭 열기**를 권장합니다.
